# Airbnb Listings Data Analysis

A comprehensive statistical analysis of Airbnb property listings using SAS programming, covering advanced data manipulation, SQL queries, and statistical reporting techniques.

## Project Overview

This project analyzes real Airbnb listings data from InsideAirbnb, containing approximately 181,928 observations. The analysis explores property characteristics, pricing patterns, host information, and review metrics across different neighborhoods.

## Dataset

- **Source:** InsideAirbnb
- **File:** Listings.csv
- **Observations:** ~181,928
- **Variables:** Listing ID, host information, room types, pricing, bedrooms, neighborhoods, review metrics

## Files

- `analysis.sas` - Complete SAS code implementation
- `STAT_506_Final_Project.pdf` - Full project report with outputs and visualizations
